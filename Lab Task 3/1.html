Lab Task 3: HTML and CSS

Home Tasks Resources

Task Overview

In this lab task, you will create a simple webpage using HTML and CSS. The page should include a header, navigation bar, main content area, and a footer. You can use the provided demo.html as a reference for structure and styling.

Instructions

  1. Create a new HTML file named "index.html".
  2. Set up the basic HTML structure with DOCTYPE, html, head, and body tags.
  3. Add a header with a title for your webpage.
  4. Create a navigation bar with links to Home, Tasks, and Resources.
  5. Add a main content area with at least two sections: an introduction and features.
  6. Style your webpage using an external CSS file named "style.css". Include styles for the header, navigation bar, main content, and footer.
  7. Test your webpage in a web browser to ensure it displays correctly.
